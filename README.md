### 01. Array
- 陣列的基本特性與記憶體配置
- 為何 Array 大小通常是固定的
- 與 Linked List 的差異比較

### 02. Linked List
- 單向鏈結串列的結構與指標操作
- 插入與刪除為何比 Array 彈性
- 時間與空間複雜度分析

### 03. Stack
- LIFO（後進先出）特性
- 使用 Array / Linked List 實作
- 常見應用：函式呼叫、括號匹配

### 04. Queue
- FIFO（先進先出）特性
- Circular Queue 的必要性
- 與 Stack 的使用情境比較

### 05. Hashing
- Hash Function 的設計概念
- Collision 與處理方法（Linear Probing）
- 平均與最差時間複雜度討論

### 06. Tree
- Binary Search Tree（BST）結構
- Tree Traversal（Inorder / Preorder / Postorder）
- BST 退化問題與平衡樹概念

### 07. Heap
- Complete Binary Tree 與 Heap Property
- Max Heap / Min Heap
- Priority Queue 與時間複雜度

### 08. Graph
- Graph 的基本定義與表示方式
- DFS / BFS 的差異與用途
- 使用 Adjacency List 實作 traversal
