## 03_Stack
- **Q1:** Stack 基本操作？  
  **AI提示回答:** Push (入棧)、Pop (出棧)、Peek (查看頂端元素)。

- **Q2:** LIFO 原理？  
  **AI提示回答:** Last In, First Out → 最後放進去的元素最先取出。

- **Q3:** Stack 應用？  
  **AI提示回答:** 函數呼叫堆疊、括號匹配、Undo/Redo。

- **Q4:** Stack 實作方式？  
  **AI提示回答:** Array 或 Linked List 都可。

- **Q5:** Stack 優點？  
  **AI提示回答:** 操作簡單，維護順序效率高。
