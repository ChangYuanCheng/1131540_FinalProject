## 02_LinkedList
- **Q1:** Singly vs Doubly Linked List 差別？  
  **AI提示回答:** Singly 只有 next 指標；Doubly 有 next + prev，可雙向遍歷。

- **Q2:** Linked List 插入/刪除時間複雜度？  
  **AI提示回答:** O(1) 如果已知節點指標；搜尋位置需 O(n)。

- **Q3:** Linked List 應用？  
  **AI提示回答:** Stack/Queue 實作、圖的鄰接表、動態記憶體管理。

- **Q4:** Linked List 優點？  
  **AI提示回答:** 動態大小，插入刪除快，節省搬移成本。

- **Q5:** Linked List 缺點？  
  **AI提示回答:** 存取慢，額外指標佔記憶體。
