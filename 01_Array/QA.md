
## 01_Array
- **Q1:** Array 與 Linked List 差別？  
  **AI提示回答:** Array 連續記憶體，存取快 O(1)，插入刪除慢 O(n)；Linked List 分散記憶體，存取慢 O(n)，插入刪除快 O(1)。

- **Q2:** Array 插入或刪除元素的時間複雜度？  
  **AI提示回答:** 平均 O(n)，因為可能要搬移其他元素。

- **Q3:** 靜態陣列 vs 動態陣列？  
  **AI提示回答:** 靜態陣列大小固定；動態陣列可自動擴充，但可能需要搬移。

- **Q4:** Array 的優點？  
  **AI提示回答:** 快速存取元素，記憶體連續，實現簡單。

- **Q5:** Array 的缺點？  
  **AI提示回答:** 插入刪除慢，大小固定（靜態陣列）。
